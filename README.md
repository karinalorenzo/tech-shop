# Tech Shop — Modernization of an Online Store

A front‑end redesign project focused on rebuilding a legacy 2010 tech e‑commerce into a modern, accessible, responsive, and semantically structured interface. The goal is to recreate the original store using current HTML5 standards, a mobile‑first approach, and a clean, contemporary visual system.

## Project Overview

This project consists of rewriting an outdated online tech shop from scratch. The original version uses obsolete practices such as XHTML Transitional, absolute positioning for layout, non-semantic `<div>` structures, and inline JavaScript-generated content. The objective is to rebuild the store using modern front‑end development principles.

The final result must resemble a contemporary e‑commerce catalog: clean, readable, responsive, and accessible across all screen sizes.

## Objectives

- Build a fully semantic HTML5 structure with proper landmarks.
- Implement a mobile‑first responsive layout.
- Apply a modern visual system based on the provided design guidelines.
- Use CSS Grid for the product catalog.
- Ensure accessibility through correct use of headings, alt text, table semantics, and focus states.
- Follow a professional Git workflow with atomic commits using Conventional Commits.
- Deploy the final version using GitHub Pages, Vercel or Netlify.

## Tech Stack

- **HTML5** — semantic structure and accessibility.
- **Tailwind CSS** — utility‑first styling following a mobile‑first approach.
- **Custom CSS variables** — design tokens for colors, typography, spacing, borders, and shadows.
- **Git & GitHub** — version control and deployment.
